¿Que es ALERTIA?

ALERTIA© es una solución creada por DEXA©, pensada para conjuntos habitacionales, la cual permite el uso de tecnologías tanto físicas como digitales para generar alarmas de sonido y baliza. Un usuario administrador designado, podrá monitorear en todo momento las activaciones generadas por los demás usuarios, pudiendo comprobar el punto exácto de activación, el sistema se encargará de llevar el registro de las actividades.

Sirena

Mediante la aplicación, los usuarios serán capaces de activar la alarma de sonido, dejando un registro tanto de la hora de activación como la ubicación del usuario.

Baliza

El usuario, también podrá generar alarmas silenciosas, mediante el uso de bálizas, lo cual generará una alerta dejando registro, de hora y ubicación en el sistema.

Ok

El usuario que haya activado alguna de las alarmas, así como el usuario administrador, podrán cancelar la alarma en todo momento. Guardando también un registro.

Geolocalización de las activaciones

Con ALERTIA© tanto el usuario administrador como los demás miembros de la red, podrán saber en todo momentos la ubicación exácta de quien activó la alarma. Para de esta manera poder elaborar un plan de acción de la mejor manera posible.

Gestion desde computador

Para el usuario administrador, la plataforma podrá ser visualizada mediante pantallas de ordenador, entregando una vista más completa y detallada.

Activaciones mediante celular

Los habitantes y usuarios de la aplicación, podrán realizar activaciones de las alarmas mediante el uso de su celular.
